# Real-Time Flight Data Analytics

## Project Overview

This project builds a Python-based ETL pipeline to extract live aircraft data from the OpenSky API, clean and transform the data, and prepare it for storage in Amazon S3 and visualization in Power BI.

## Project Objectives

- Extract live aircraft data from the OpenSky API
- Clean and transform raw flight data using Python and Pandas
- Store processed datasets as CSV files
- Upload processed data to Amazon S3
- Build an interactive Power BI dashboard
- Analyze altitude, velocity, aircraft status, origin countries, and air traffic activity

## Technologies Used

- Python
- Pandas
- Requests
- AWS S3
- Boto3
- Power BI
- GitHub
- OpenSky API

## Project Architecture

OpenSky API → Python ETL → CSV Files → Amazon S3 → Power BI Dashboard
